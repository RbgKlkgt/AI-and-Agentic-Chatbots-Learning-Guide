"# Minimalist FastAPI Client API

This is a simple FastAPI application for managing clients using a CSV database.

## Setup
1. Activate the virtual environment: `.\venv\Scripts\Activate.ps1` (PowerShell) or `call venv\Scripts\activate` (cmd).
2. Install dependencies: `pip install fastapi uvicorn` (already done).
3. Run the server: `python run.py`.

The server runs at `http://127.0.0.1:8000`.
